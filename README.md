#  Hospital Emergency Room Dashboard (Microsoft Excel)

## Overview
This project is an interactive **Hospital Emergency Room Dashboard** developed in **Microsoft Excel**. The dashboard provides a clear overview of hospital emergency room performance using interactive charts, KPIs, and slicers. It helps analyze patient data and supports better decision-making through meaningful visual insights.

---

##  Objectives
- Analyze emergency room patient data.
- Monitor hospital performance using key metrics.
- Create an interactive and user-friendly dashboard.
- Gain insights into patient demographics and hospital operations.

---

##  Dashboard Features
-  Total Number of Patients
-  Average Wait Time
-  Patient Satisfaction Score
-  Admission Status
-  Gender-wise Patient Analysis
-  Age Group Distribution
-  Department Referral Analysis
-  Interactive Month & Year Filters

---

##  Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Data Cleaning
- Data Visualization

---

##  Key Insights
- Track the total number of patients visiting the emergency room.
- Analyze average waiting time for patients.
- Monitor patient satisfaction levels.
- Compare admitted and non-admitted patients.
- Understand patient distribution by gender and age group.
- Identify department-wise patient referrals.
- Filter and analyze data by month and year.
